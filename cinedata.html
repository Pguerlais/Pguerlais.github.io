02

Moteur de recommandation

Approche hybride
& similarité cosinus

Le système combine des features hétérogènes — texte, catégories et variables numériques — pour produire un vecteur de représentation dense par film. L'algorithme NearestNeighbors de scikit-learn calcule ensuite la distance cosinus entre ce vecteur et l'ensemble du catalogue.

Ce choix permet de gérer efficacement les matrices creuses (sparse) issues du TF-IDF, tout en assurant des performances rapides en production — même sur un catalogue de plusieurs centaines de milliers de films.

Les recommandations croisent ainsi genre, casting, réalisateurs, popularité, année et description textuelle pour identifier des films genuinement similaires.

# Preprocessing hybride vectorizer = TfidfVectorizer(max_features=5000) text_features = vectorizer.fit_transform(df['text_combined']) mlb = MultiLabelBinarizer() cat_features = mlb.fit_transform(df['genres']) scaler = StandardScaler() num_features = scaler.fit_transform(df[['popularity', 'year']]) # Matrice finale sparse X = hstack([text_features, cat_features, num_features]) # Moteur de recommandation model = NearestNeighbors(metric="cosine", algorithm="brute") model.fit(X)
